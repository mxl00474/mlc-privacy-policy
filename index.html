日本語版

1. はじめに

MultiLingual Chat（以下「本アプリ」）は、ユーザーの英会話練習を支援するモバイルアプリです。 本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように利用するかを説明します。

2. 収集するデータの種類

本アプリは以下のデータを収集します。

データ種別 収集内容 収集タイミング
メールアドレス Google / Apple アカウントのメールアドレス ログイン時
音声データ マイクで録音した会話音声 音声会話セッション中
会話テキスト AI との会話のテキスト内容 会話セッション中
会話要約 セッション終了時に AI が生成した会話の要点メモ 会話セッション終了時
設定情報 表示設定・入力モード等のアプリ設定 設定変更時

3. データの利用目的

収集したデータは以下の目的で利用します。

  • 英会話練習サービスの提供（AI との対話機能）
  • 過去の会話内容にもとづくパーソナライズされた会話体験の提供（クロスセッション記憶）
  • ユーザー認証・アカウント管理
  • アプリ設定の保存・復元
  • サービスの改善・品質向上

本アプリは、収集したデータを上記以外の目的で利用しません。

4. OpenAI へのデータ送信について

本アプリは、英会話 AI 機能の提供のために OpenAI, Inc. のサービス（Chat Completions API / Realtime API）を利用しています。

ユーザーが音声会話またはテキスト会話を行う際、以下のデータが OpenAI のサーバーに送信されます。

  • 録音した音声データ（音声入力モード使用時）
  • 入力したテキストメッセージ
  • 会話の文脈（直前のやり取り）

また、会話セッション終了時に、次回以降の会話体験を向上させるため、当該セッションの会話内容を要約する目的でも OpenAI の Chat Completions API にデータが送信されます。生成された要約はサーバーに保存され、次回の会話開始時に AI の記憶として活用されます。

OpenAI のデータ取り扱いについては OpenAI Privacy Policy をご参照ください。

5. データの保存期間と削除方法

データ種別 保存場所 保存期間 削除方法
メールアドレス・アカウント情報 サーバー（Cloud SQL） アカウント削除まで アプリ内のアカウント削除機能から削除可能
会話テキスト サーバー（Cloud SQL） アカウント削除まで アカウント削除と同時に削除
会話要約 サーバー（Cloud SQL） アカウント削除まで アカウント削除と同時に削除。個別削除はお問い合わせから申請可能
音声データ OpenAI サーバー OpenAI のポリシーに従う OpenAI のデータ削除リクエストフォームから申請
アプリ設定 デバイス内（AsyncStorage） アプリアンインストールまで アプリをアンインストールすることで削除

6. 第三者への情報提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく開示が必要な場合
  • サービス提供に必要なサービスプロバイダー（OpenAI 等）への提供

7. 未成年者のプライバシー

本アプリは 13 歳未満の方を対象としていません。 13 歳未満の方が個人情報を提供していることが判明した場合、速やかに削除します。

8. プライバシーポリシーの変更

本ポリシーは必要に応じて更新される場合があります。 重要な変更がある場合は、アプリ内通知またはメールでお知らせします。

English Version

1. Introduction

MultiLingual Chat (the “App”) is a mobile application designed to help users practice English conversation. This Privacy Policy explains what information the App collects and how it is used.

2. Types of Data Collected

The App collects the following data:

Data Type Content When Collected
Email address Email address from Google / Apple account At login
Audio data Voice recordings from the microphone During voice conversation sessions
Conversation text Text content of conversations with AI During conversation sessions
Conversation summary AI-generated key points from the session At the end of each conversation session
App settings Display settings, input mode, and other preferences When settings are changed

3. How We Use Your Data

Collected data is used for the following purposes:

  • Providing the English conversation practice service (AI interaction features)
  • Providing a personalized conversation experience based on past conversation history (cross-session memory)
  • User authentication and account management
  • Saving and restoring app settings
  • Service improvement and quality enhancement

The App does not use collected data for any purposes beyond those listed above.

4. Data Transmission to OpenAI

The App uses services from OpenAI, Inc. (Chat Completions API / Realtime API) to provide AI conversation features.

When a user engages in voice or text conversation, the following data is transmitted to OpenAI’s servers:

  • Recorded audio data (when using voice input mode)
  • Typed text messages
  • Conversation context (recent exchanges)

In addition, at the end of each conversation session, the session’s conversation content is also sent to OpenAI’s Chat Completions API for the purpose of generating a summary to improve future conversation experiences. The generated summary is stored on our servers and used at the start of subsequent conversations as the AI’s memory.

For OpenAI’s data handling practices, please refer to the OpenAI Privacy Policy.

5. Data Retention and Deletion

Data Type Storage Location Retention Period How to Delete
Email / Account information Server (Cloud SQL) Until account deletion Delete via in-app account deletion feature
Conversation text Server (Cloud SQL) Until account deletion Deleted when account is deleted
Conversation summary Server (Cloud SQL) Until account deletion Deleted when account is deleted. Individual deletion requests can be submitted via our contact form
Audio data OpenAI servers Subject to OpenAI’s policy Request deletion via OpenAI’s data deletion form
App settings Device (AsyncStorage) Until app uninstall Deleted by uninstalling the app

6. Sharing with Third Parties

The App does not share your personal information with third parties except in the following cases:

  • When you have given consent
  • When required by law
  • With service providers necessary for delivering the service (e.g., OpenAI)

7. Children’s Privacy

The App is not intended for children under the age of 13. If we become aware that a child under 13 has provided personal information, we will delete it promptly.

8. Changes to This Policy

This policy may be updated as needed. For significant changes, we will notify you via in-app notification or email.
